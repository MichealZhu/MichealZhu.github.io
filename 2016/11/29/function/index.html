闭包的简单理解

闭包

就对闭包的理解来说始于和函数内部的值外部无法直接获取 （单纯的获取方式：函数不调用不执行 1 调用 2 调用产生值 return将其送出）而函数内部可以获取 上下作用域 所以return 可以获取值
那么需要将值进行修改呢 可以用函数 所以return （出函数 函数里操作将值进行修改 将值return）
所以闭包的基本形式：一个外部函数和一个内部函数（return一个内部函数） 通过内部函数来访问更改外部函数的值
function test(){
var a= 1;
return function(){
return a=a+1;
}
}
闭包的作用：给以一个安全的环境封闭的空间 能够让闭包中的值不会轻易改变（在全局中随意可以更改）和污染全局 需要通过给以的特定的方法获取或改变 （作用域的关系）

eg function test(){
var name=’zjl’;
}
test（）；此刻值已存在 只不过外部无法获取

如进行闭包的使用 则可以进行值的获取和更改 （想要更改我的数据必须通过我的方法 也可在此之中进行逻辑的校验）
function test （）{
var name =’zjl’;
return {
getname: function(){
return name;(name在外部函数中获取)，
setname:function(a){
if(a==’zjl’){
throw{
‘it is a mistake’
}
}
return name=a;
}
}
}
}
